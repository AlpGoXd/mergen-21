## File Naming Convention

```
<part-name>_drawing_v<version>.pdf
```

Example: `horn-antenna_drawing_v1.pdf`

## Export Notes

- Export from Inventor: **File → Export → PDF** (Drawing environment)
- Include: all standard views (front, top, side, isometric), section views where needed
- Sheet size: A3 or A4
- Title block: part name, revision, author, date, material, scale
